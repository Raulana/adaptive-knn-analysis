# ACKER Adaptive kNN Classification Project

This project explores the ACKER (Adaptive Classifier based on kNN Expected Accuracy) framework.

The goal of the project is to improve traditional kNN classification by dynamically selecting the number of neighbors based on expected accuracy.

## Topics Covered

- k-Nearest Neighbors (kNN)
- Adaptive classification
- Similarity functions
- Expected accuracy
- Classification evaluation

## Project Materials

This repository includes:
- Project report
- Presentation slides
- Experimental analysis

## Learning Outcomes

Through this project, I learned:
- How adaptive kNN algorithms work
- The role of similarity functions
- Performance comparison between classifiers
- Evaluation of machine learning models
